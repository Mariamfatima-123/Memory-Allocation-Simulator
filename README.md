# 💻 Memory Allocation Simulator (C++)

## 📌 Description
This project simulates memory allocation techniques used in Operating Systems. It demonstrates how processes are allocated to memory blocks using different strategies.

---

## 🚀 Features
- First Fit Algorithm
- Best Fit Algorithm
- Worst Fit Algorithm
- Fragmentation Calculation
- Menu-driven Console Application

---

## 🧠 Algorithms

### First Fit
Allocates process to the first suitable block.

### Best Fit
Allocates process to the smallest suitable block.

### Worst Fit
Allocates process to the largest available block.
